My Work

A collection of projects I've worked on.

Spam Email Classifier

Engineered 30+ text-based features and built supervised ML models (including logistic regression) with cross-validation to improve accuracy by ~15% over a baseline.

Housing Prices Predictor (Cook County)

Built end-to-end linear regression pipelines to predict log sale prices, analyzed residuals/model errors, and reflected on fairness and bias in property tax assessment modeling.

NGram TimeSeries Engine

Implemented core Java data structures for an N-gram exploration tool (TimeSeries + NGramMap) with efficient time-range slicing, aggregation, and relative-frequency queries over large corpora.

Hotel AI Email Assistant

Built a hotel-focused AI assistant to read, classify, and draft replies to guest emails with human review, using multi-agent workflows and secure OAuth2 + Microsoft Graph/Outlook API access.

Jeopardy Wagering Behavior Analysis

Analyzed Daily Double and Final Jeopardy wagers by computing wager-to-score ratios and stratifying by game state; used SQL (pgAdmin 4) and visualizations to quantify strategic risk-taking.
